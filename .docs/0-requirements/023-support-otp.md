Dựa trên requirement .docs/0-requirements/017-support-a2ui-protocol.md và .docs/0-requirements/018-support-dynamic-frontend-components.md, chúng ta đã có hỗ trợ render các UI components một cách động (dynamically) về frontend.

Bây giờ mình sẽ hướng đến render một nhóm các block, theo chủ đề, ví dụ OTP. Hãy xem các ví dụ ở đây:

- https://ui.shadcn.com/blocks/otp
- Hoặc ở đây: .docs/0-requirements/images/otp-sample.png

Hãy xây dựng một a2ui block component cho otp và có thể render về frontend cho block component này.

Block Component này sẽ là một a2ui tool được hỗ trợ trong cả a2ui agent và a2ui agent with loop.
