Artifact pannel dùng cho mục đích tương tác với chat, nên hạn chế một số tính năng của BlockNoteView. Ví dụ:

- Disable context menu mặc định của BlockNoteView
- Thay thế bằng một context menu theo chiều ngang mới, với menu item: Chat with agent.
- Menu này xuất hiện khi user chọn một đoạn text nào đó. Khi user nhấn nút "Chat with agent", sẽ tô màu đoạn text đó màu vàng, và focus vào ô chat để user có thể bắt đầu chat.
