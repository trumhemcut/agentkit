Trong canvas mode, không gian hiển thị của chat container khá nhỏ bé. Nên tối ưu vùng không gian này tốt nhất. 

Do đó: 

### đối với agent message 

- Không cần hiển thị icon
- Hiển thị content của agent trọn vẹn trong khung không gian của agent message

### đối với human message 

- Không cần hiển thị icon
- Giữ nguyên format content
