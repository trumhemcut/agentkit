Build the frontend skeleton for Agentic AI solution using NextJS, Shadcn, and AG-UI. AG-UI will be the main protocol for agent <-> frontend communication. Just build a simple chat page with the chatbox and the chat history window.

The layout
- Header
- Left menu to create new chat and chat threads
- Main content with chatbox and chat history between agents and human
- Agent & human has a beautiful svg icons, icons should be from lucide react icons.

* use context7
* the library of ag-ui for context7 is here: https://context7.com/ag-ui-protocol/ag-ui
* @ag-ui/client is the offcial package of ag-ui for front-end
* using localstorage for thread persistence
